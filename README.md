<p align="center"><b>프로젝트 특징</b></p>
<ul>
  <li>
    주소 API, 결제 API를 이용한 간편 시스템 구현
  </li>
  <li>
    이용자가 원하는 날짜, 시간에 맞춰 좌석 선택 가능, 좌석의 등급에 따라 합리적인 가격 구성.
